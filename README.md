# SOLUTION DIGIYLYFE

**La solution existe. Maintenant, il faut la rendre visible.**

Ce dépôt porte la page publique **SOLUTION DIGIYLYFE** : une page manifeste courte, visuelle et terrain autour de la métaphore de l’oasis dans le désert numérique.

DIGIYLYFE est présenté comme une source numérique qui appartient au terrain : loueurs, vendeurs, chauffeurs, bâtisseurs, artisans, services locaux et professionnels indépendants.

---

## Vision

Dans le désert numérique, beaucoup de professionnels cherchent de la visibilité, du contact direct et une vraie indépendance économique.

DIGIYLYFE ne vient pas remplacer le terrain. DIGIYLYFE vient l’organiser, le rendre visible et lui laisser la main.

Principes clés :

- **0% commission**
- **paiement direct au professionnel**
- **QR direct**
- **contact direct**
- **fiches métiers visibles**
- **visibilité locale et mondiale**
- **outils simples pour les métiers du terrain**

---

## Métiers ciblés

La page parle en priorité aux métiers qui ont besoin de visibilité directe :

- **Loueurs** : logements, chambres, villas, hébergements, réservations.
- **Vendeurs** : boutiques, produits, market local, commandes.
- **Chauffeurs** : trajets, AIBD, Saly, Dakar, mobilité locale.
- **Bâtisseurs** : artisans, construction, dépannage, services terrain.

## Déploiement GitHub Pages

1. Créer un dépôt nommé :

```txt
solution-digiylyfe
```

2. Ajouter les fichiers :

```txt
index.html
README.md
```

3. Optionnel : ajouter le clip MP4 dans le dépôt :

```txt
digiylyfe-oasis-clip-CORRIGE-lisible-son-booste.mp4
```

4. Activer GitHub Pages :

```txt
Settings → Pages → Deploy from a branch → main → /root
```

5. L’URL sera de type :

```txt
https://<compte-github>.github.io/solution-digiylyfe/
```
